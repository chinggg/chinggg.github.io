GuessKey

题意:随机生成大数,以十进制呈现,但后续变换操作都在二进制下完成,然后自行输入mask值作为变换的一个参数,变换只对key上闭区间[p,q]进行,就是用mask的一位(依次取位,到尾则循环)和原key一位异或后的值代替原值,连续3次正确输入变换后的key即得flag。
解法:由于p,q随机且未知,所以该段变换必须保持原值不变,使变换后key不变,结合异或的特性只需令mask为0

from random import randint
import os
#from flag import flag
N=64
key=int(raw_input("key:"))
#key=randint(0,2**N)
#print key
key=bin(key)[2:].rjust(N,'0')
print "bin_key:"+key
count=0
while True:
	p=0
	q=0
	new_key=''
	zeros=[0]
	for j in range(len(key)):
		if key[j]=='0':
			zeros.append(j)
	p=zeros[randint(0,len(zeros))-1]
	q=zeros[randint(0,len(zeros))-1]
	print "p=",p,"q=",q
	try:
		mask=int(raw_input("mask:"))
	except:
		exit(0)
	mask=bin(mask)[2:]
	if p>q:
		tmp=q
		q=p
		p=tmp
	cnt=0
	for j in range(0,N):
		if j in range(p,q+1):
			new_key+=str(int(mask[cnt])^int(key[j]))
		else:
			new_key+=key[j]
		cnt+=1
		cnt%=len(mask)
	key=new_key
	print "new_key:"+key
	try:
		guess=int(raw_input("guess:"))
	except:
		exit(0)
	if guess==int(key,2):
		count+=1
		print 'Nice.'
	else:
		count=0
		print 'Oops.'
	if count>2:
		print flag
